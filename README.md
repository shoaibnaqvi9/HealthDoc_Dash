# HealthDoc_Dash - Doctor Consultation System
HealthDoc is a doctor consultation web application built with ASP.NET Core and SQL Server, designed to manage patients, doctors, appointments, and medical reports. It features a clean dashboard with analytics, efficient appointment handling, and a scalable architecture suitable for real-world healthcare systems.

This project is designed for learning, demonstration, and real-world extensibility.

# Features
## Doctor Management
- Register and manage doctors
- Specialty-wise doctor listing
- View doctor appointments

## Patient Management
- Patient registration and profile management
- View appointment history
- Store basic medical details

## Appointment Scheduling
- Book appointments with doctors
- Track appointment status (Scheduled, Completed, Cancelled)
- Link appointments with medical reports

## Medical Reports
- Create reports per appointment
- Store diagnosis and prescriptions
- View patient medical history

## Dashboard & Analytics
- Total patients, doctors, appointments, and reports
- Monthly patient growth chart
- Consultation statistics
- Recent appointments overview
